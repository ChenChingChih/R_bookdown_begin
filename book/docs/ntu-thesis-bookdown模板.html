6.1 安裝

最新版

#install.packages('remotes')
remotes::install_github("liao961120/ntuthesis")

套件所需的字體檔

#install.packages('remotes')
remotes::install_github("liao961120/ntuthesis")

6.2 匯入論文模板

打開ntuthesis.Rproject在 console 執行：

ntuthesis::import_template("project_name")

會出現project_name資料夾。
在project_name資料夾中開一個R-project檔，然後開啟該R-project檔。

6.2.1 編輯封面

按照原作者指令操作，會安裝.tex需要用到的套件到tinytex中。

6.2.2 Compile 論文

將01-install-compile.rmd內的28~34行刪除，

if (knitr::is_html_output()) {
  knitr::include_graphics("figs/rmd-template.gif")
} else {
  knitr::include_graphics("figs/rmd-template.png")
}

調整_output.yml 5~6行，

  includes:
#    in_header: latex/custom.js
#    after_body: disqus.html

再點選build book即可生成gitbook和pdf檔。

bookdown::render_book("index.Rmd", "bookdown::gitbook")
bookdown::render_book("index.Rmd", "bookdown::pdf_book")
