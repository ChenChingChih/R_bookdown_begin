Chapter 5 使用狀況追蹤

了解你的影響力

透過Goolge Analytics來追蹤書本的使用狀況， 先申請Goolge Analytics的追蹤碼， 再將追蹤碼鑲嵌至電子書中。(Marwick 2016)
在_output.yml加入includes: in_header: [ google_analytics.html]，

bookdown::gitbook:
  includes:
    in_header: [ google_analytics.html]

接著新開一個google_analytics.html，將Goolge Analytics的代碼放到google_analytics.html裡。

#從這裡以下開始複製你的追蹤碼到google_analytics.html裡  
<!-- Global site tag (gtag.js) - Google Analytics -->
<script async src="https://www.googletagmanager.com/gtag/js?id=UA-159991960-1"></script>
<script>
你的追蹤
</script>
